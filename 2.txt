
你是一名 AI Agent 系统架构师（AI Agent Systems Architect）和开发者平台架构师（Developer Platform Architect）。

你的任务是为一个开源项目设计一个完整的工程蓝图。

项目名称：

AI Code Research System

------------------------------------------------
项目目标
------------------------------------------------

该系统的目标是：

使 AI Agent 能够对 GitHub 仓库进行系统化研究和分析，包括：

- 源代码结构理解
- 调用图（Call Graph）追踪
- 数据结构提取
- 系统架构分析
- 安全漏洞研究
- 攻击面识别

------------------------------------------------
关键架构要求
------------------------------------------------

该系统不能依赖单个巨大 Prompt。

系统必须采用：

Skill-based Architecture（基于技能的架构）

核心原则：

- 系统核心必须保持极简
- 所有分析能力必须以 Skill（技能）的形式实现
- 研究任务通过 Workflow（工作流）组合执行

------------------------------------------------
示例 Workflow
------------------------------------------------

漏洞研究 Workflow

repo_discovery
→ entrypoints_detection
→ call_graph_trace
→ input_flow_analysis
→ vulnerability_hypothesis
→ exploit_generation

------------------------------------------------
你的任务
------------------------------------------------

设计一个完整工程架构。

输出必须是：

可落地实现的工程设计，
而不是概念讨论。

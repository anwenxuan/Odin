## 分析任务

### 1. 网络攻击面总览

| 攻击面类型 | 入口点 | 端口/地址 | 协议 | 风险等级 | 备注 |
|-----------|--------|----------|------|---------|------|
| TCP 监听 | 0.0.0.0:6379 | 6379 | RESP | 极高 | Redis 默认无认证 |
| Unix Socket | /tmp/redis.sock | - | Unix | 中 | 本地访问 |
| HTTP API | 0.0.0.0:8080 | 8080 | HTTP | 高 | REST API |
| CLI | STDIN | - | Text | 中 | 本地攻击面 |
| 管理接口 | 127.0.0.1:16379 | 16379 | RESP | 高 | Sentinel/Cluster Bus |

### 2. 协议层攻击面

**协议格式分析**（对自定义协议尤其重要）：

| 协议名 | 版本 | 帧格式 | 编码方式 | 攻击面 |
|--------|------|--------|---------|--------|
| RESP | 2/3 | <type><length><data> | 文本 | 协议解析器漏洞 |
| 自定义 | v1 | 二进制 | Binary | 解析器缓冲区溢出 |

**协议解析路径追踪**：
```
网络数据 → recv() → 输入缓冲区 → 协议解析器 → 命令分发 → 处理逻辑
     ↑
     漏洞可能在这里（缓冲区溢出、解析逻辑错误）
```

### 3. 命令/消息类型攻击面

| 命令类型 | 处理函数 | 输入验证 | 危险操作 | 风险 |
|---------|---------|---------|---------|------|
| CONFIG SET | configCommand | 弱 | 写文件系统 | 极高 |
| DEBUG SEGFAULT | debugCommand | 无 | 崩溃 | 高 |
| EVAL | evalCommand | Lua 沙箱 | 执行任意代码 | 极高 |

### 5. 攻击向量映射

| 攻击向量 | 入口点 | 触发条件 | 利用难度 | 影响程度 | 风险等级 |
|---------|--------|---------|---------|---------|---------|
| 协议解析器 DoS | 6379 | 发送畸形 RESP 数据 | 低 | 服务崩溃 | 严重 |
| 未授权命令执行 | 6379 | 无认证或弱密码 | 低 | RCE/数据泄露 | 严重 |
| CONFIG 注入 | 6379 | 认证后可写配置 | 低 | 服务器控制 | 严重 |
| Lua 沙箱逃逸 | 6379 | EVAL 命令 | 中 | 沙箱逃逸 | 高 |
| 缓冲区溢出 | 6379 | 超长命令参数 | 高 | RCE | 严重 |
| 资源耗尽 | 6379 | 大量连接/键 | 低 | DoS | 高 |

### 6. 攻击路径示例

追踪从外部到内部的典型攻击路径：

**路径 1：无认证 Redis → 服务器控制**
```
攻击者 → 连接 6379 端口（无密码）
  → CONFIG SET dir /tmp
  → CONFIG SET dbfilename evil.rdb
  → SAVE
  → (RDB 文件被写入 /tmp/evil.rdb，覆盖 crontab 等计划任务)
  → 反弹 Shell
```

**路径 2：Lua 沙箱逃逸**
```
认证用户 → EVAL "require('os')" 0
  → Lua 沙箱未禁用 os 库
  → os.execute('id')
  → 服务器代码执行
```

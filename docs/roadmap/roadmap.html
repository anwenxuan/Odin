Development Plan

Roadmap

From foundational architecture to production-grade autonomous vulnerability research — every milestone on the path to v1.0.

Completed

v0.1.0

Foundation

January 2024
Tool Executor

Pluggable execution engine with timeout, retry, and structured output across all 12 built-in skills.

January 2024
12 Built-in Skills

Grep, AST, Semgrep, SBOM, CVSS, CWE, Deps, License, Secrets, Malware, Fuzz, CVE Lookup — ready to orchestrate.

January 2024
CLI Interface

Command-line interface with workflow selection, target specification, and verbose output modes.

January 2024
YAML Workflow Definitions

Human-readable workflow configs defining skill sequences, parameters, and pass-through logic.

January 2024
Mock Provider

Offline testing mode using a local mock LLM provider — no API keys required to get started.

Completed

v0.2.0

Agent Core

June 2024
Agent Loop

ReAct-style reasoning engine with thought/action/observation cycles, state machine, and configurable max iterations.

June 2024
DAG Parallel Execution

Layered task graph with topological scheduling — independent tasks run in parallel across DAG layers.

June 2024
MCP Integration

Model Context Protocol support enabling dynamic tool discovery and external resource connections.

June 2024
FastAPI REST Server

Production-grade API for workflow management, session persistence, and programmatic access.

June 2024
RAG Knowledge Store

Vector-backed retrieval for persisting vulnerability findings, CVE records, and cross-session research context.

In Progress

v0.3.0

Stability

Q4 2024 (Target)
Checkpoint & Resume

Agent state snapshots at configurable intervals — recover any research session after interruption without losing progress.

Q4 2024 (Target)
Docker Sandbox

Containerized tool execution with resource limits, network isolation, and clean-up guarantees.

Q4 2024 (Target)
Evaluation Benchmarks

SWE-bench integration, real-world CVE datasets, and quantitative precision/recall metrics for agent quality.

Planned

Scale

2025 (Target)
Multi-Agent Orchestration

Permission System

Cost & Token Budget

Setting Up Your Environment

Copy the example environment file and configure it for your setup:

terminal
cp .env.example .env
.env
# ─── LLM Provider Configuration ───────────────────────────
# Choose ONE provider — set API key for that provider:

# OpenAI (recommended: gpt-4o-mini for cost efficiency)
OPENAI_API_KEY=sk-...

# OR Anthropic Claude
# ANTHROPIC_API_KEY=sk-ant-...

# ─── Provider & Model Selection ─────────────────────────────
ODIN_PROVIDER=openai
ODIN_MODEL=gpt-4o-mini

# ─── Model Parameters ──────────────────────────────────────
ODIN_TEMPERATURE=0.0
ODIN_MAX_TOKENS=4096

# ─── Workflow Configuration ────────────────────────────────
MAX_CALL_GRAPH_DEPTH=8
WORKFLOW_TIMEOUT_SEC=300
MAX_ITERATIONS_PER_STEP=20

# ─── Persistence ──────────────────────────────────────────
EVIDENCE_STORE_PATH=./data/evidence
MEMORY_STORE_PATH=./data/memory
RAG_PERSIST_DIR=./data/rag

Environment Variables

VariableDefaultDescription
OPENAI_API_KEY — OpenAI API key. Required if ODIN_PROVIDER=openai
ANTHROPIC_API_KEY — Anthropic API key. Required if ODIN_PROVIDER=anthropic
ODIN_PROVIDER openai LLM provider: openai, anthropic, ollama, mock
ODIN_MODEL gpt-4o-mini Model name. See model recommendations below.
ODIN_TEMPERATURE 0.0 LLM temperature (0.0–1.0). Use 0.0 for deterministic code analysis.
ODIN_MAX_TOKENS 4096 Maximum tokens in LLM response.
MAX_CALL_GRAPH_DEPTH 8 Maximum depth for call graph tracing.
WORKFLOW_TIMEOUT_SEC 300 Overall timeout per workflow (seconds).
MAX_ITERATIONS_PER_STEP 20 Max LLM → tool iterations per SkillAgent step.
EVIDENCE_STORE_PATH ./data/evidence Path to SQLite evidence database.
MEMORY_STORE_PATH ./data/memory Path to SQLite memory database.
RAG_PERSIST_DIR ./data/rag Path to SQLite FTS5 RAG index directory.

LLM Provider Configuration

OpenAI

OpenAI's GPT-4o and GPT-4o-mini are the recommended providers for most use cases. gpt-4o-mini offers the best cost-to-performance ratio for code analysis.

ModelSpeedCostBest For
gpt-4oMediumHighComplex reasoning, large codebases
gpt-4o-miniFastLowRecommended default
gpt-4-turboMediumMediumLegacy support
.env
export OPENAI_API_KEY=sk-...
export ODIN_PROVIDER=openai
export ODIN_MODEL=gpt-4o-mini
export ODIN_TEMPERATURE=0.0
export ODIN_MAX_TOKENS=4096

Anthropic Claude

Claude models excel at long-context analysis and can handle large codebases in a single context window.

ModelSpeedContextBest For
claude-sonnet-4-20250514Fast200KRecommended default for Claude
claude-3.5-sonnet-20241022Fast200KLong codebase analysis
claude-3-opusSlow200KMaximum reasoning depth
.env
export ANTHROPIC_API_KEY=sk-ant-...
export ODIN_PROVIDER=anthropic
export ODIN_MODEL=claude-sonnet-4-20250514
export ODIN_MAX_TOKENS=4096

Ollama (Local)

Run Odin entirely locally using Ollama. No API keys or internet required.

.env
export ODIN_PROVIDER=ollama
export ODIN_MODEL=llama3
# Or any Ollama model:
# export ODIN_MODEL=codellama
# export ODIN_MODEL=mistral
# export ODIN_MODEL=qwen2.5-coder
ℹ

Ensure Ollama is running locally: ollama serve (usually runs automatically when installed). The default Ollama endpoint is http://localhost:11434/v1.

Mock Mode (No API Key)

Use the Mock adapter for testing without any API costs. Returns simulated tool calls and responses.

.env
export ODIN_PROVIDER=mock
# No API key needed for mock mode

Programmatic LLM Configuration

Configure LLM adapters directly in Python code:

config_llm.py
from agent.llm_adapter import create_adapter

# OpenAI (default)
llm = create_adapter(
    "openai",
    default_model="gpt-4o-mini",
    default_temperature=0.0,
    default_max_tokens=4096,
    # Optional: custom base URL for proxies or compatible APIs
    # base_url="https://api.openai.com/v1",
)

# Anthropic
llm = create_adapter(
    "anthropic",
    default_model="claude-sonnet-4-20250514",
    default_max_tokens=4096,
)

# Ollama (local)
llm = create_adapter(
    "ollama",
    base_url="http://localhost:11434/v1",
    default_model="llama3",
    api_key="ollama",  # Ollama doesn't need real keys
)

# Mock (testing)
llm = create_adapter("mock")

Agent Loop Configuration

The LoopConfig controls how the AgentLoop executes skill analysis:

agent/state.py
from agent.state import LoopConfig

loop_config = LoopConfig(
    # Max LLM → tool iterations per SkillAgent run
    max_iterations=20,

    # Require every conclusion to cite MEU evidence_refs
    evidence_required=True,

    # Require the final LLM output to be valid JSON
    # Falls back to text extraction if False
    require_final_json=True,

    # Continue workflow if individual steps fail
    # When False, first failure aborts the entire workflow
    allow_fallback_on_error=True,

    # Print verbose tool execution logs
    verbose=False,

    # Per-step timeout in seconds (used by ThreadPoolExecutor)
    timeout_per_step_sec=120,
)
ConfigDefaultDescription
max_iterations20Max iterations before AgentLoop gives up on a skill
evidence_requiredtrueEnforce MEU evidence citations in conclusions
require_final_jsontrueReject non-JSON final outputs
allow_fallback_on_errortrueContinue on step failure instead of aborting
verbosefalsePrint detailed execution logs
timeout_per_step_sec120Timeout per pipeline step

Pipeline Configuration

The PipelineExecutor controls parallel execution of workflow steps:

core/pipeline_executor.py
from core.pipeline_executor import PipelineExecutor

executor = PipelineExecutor(
    skill_registry=registry,
    llm_adapter=llm,
    tool_executor=tool_executor,
    evidence_store=evidence_store,
    memory_store=memory_store,
    loop_config=loop_config,

    # Maximum concurrent threads per layer
    # Default: 4
    # Set to number of CPU cores for max parallelism
    max_workers_per_layer=4,
)
ℹ

Performance tip: Set max_workers_per_layer to the number of available CPU cores for optimal parallel step execution.

Storage Configuration

EvidenceStore

Stores MEUs (Minimum Evidence Units) with code citations.

memory/evidence_store.py
evidence_store = EvidenceStore(
    persist_path=Path("./data/evidence")
)
# Uses SQLite under the hood
# Auto-creates directory if missing

MemoryStore

Stores ResearchArtifacts and Conclusions.

memory/memory_store.py
memory_store = MemoryStore(
    persist_path=Path("./data/memory")
)
# Stores artifacts with evidence cross-references
# Supports querying by skill, run, or tag

RAGStore

SQLite FTS5 full-text search over reports.

rag/store.py
rag = RAGStore(
    persist_dir=Path("./data/rag"),
    # Chunk size for text splitting
    chunk_size=512,
    # Number of relevant chunks to retrieve
    top_k=5,
)

CLI Override

Override storage paths via environment variables:

.env
EVIDENCE_STORE_PATH=/var/odin/evidence
MEMORY_STORE_PATH=/var/odin/memory
RAG_PERSIST_DIR=/var/odin/rag

API Server Configuration

The FastAPI server can be configured via environment variables or command line:

terminal
# Basic server start
python -m uvicorn cli.commands.serve:app --reload --port 8080

# With host binding
python -m uvicorn cli.commands.serve:app --host 0.0.0.0 --port 8080

# With workers (production)
python -m uvicorn cli.commands.serve:app --workers 4 --port 8080
ℹ

API server inherits all LLM configuration from environment variables. Set OPENAI_API_KEY or ANTHROPIC_API_KEY before starting the server.

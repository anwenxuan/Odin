Most existing AI agent frameworks focus on orchestration, developer automation, or general multi-agent collaboration.

Odin is designed specifically for systematic vulnerability research. It introduces a structured evidence model (MEU), deterministic research workflows, and DAG-based execution to ensure reproducible analysis.

Instead of acting as a generic agent runtime, Odin operates as a research system that enforces evidence collection, skill-driven analysis, and measurable evaluation of AI-assisted security research.

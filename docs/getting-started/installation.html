Get Started / Installation

Installation

Get Odin running in minutes. Supports pip, uv, and source installation.

System Requirements

Before installing, make sure your environment meets these prerequisites.

Python 3.11+

Required. Install via python.org or pyenv.

Git

Required for source installation. brew install git or apt install git.

Docker (optional)

Required only for sandbox isolation mode. Get Docker.

Method 1: pip (Recommended)

The easiest way to install Odin. Works with any Python 3.11+ environment.

bash 
pip install odin-agent
Using a virtual environment?
It is recommended to use a virtual environment (venv or conda) to avoid dependency conflicts.

Method 2: Source (Development)

Install from source for the latest features, or if you want to contribute to Odin.

bash 
# Clone the repository
git clone https://github.com/anwenxuan/Odin
cd Odin

# Install in editable mode with dev dependencies
pip install -e ".[dev]"
Editable install
The -e flag installs the package in editable mode, so changes to the source code take effect immediately without reinstalling.

Method 3: uv (Fastest)

uv is a blazing-fast Python package manager written in Rust. Use it for the quickest install.

bash 
# Install globally (requires uv)
uv pip install odin-agent

# Or within project
uv pip install -e "."

Verify Installation

After installation, verify everything is working by checking the version.

bash 
$ odin --version
odin v0.2.0

Common Installation Issues

"command not found: odin" or "No module named odin"

Make sure your Python environment is on your PATH. Try running with the full path: 

python -m odin --version

If this works, add export PATH="$PATH:$HOME/Library/Python/X.X/bin" to your shell config. Replace X.X with your Python version.

Dependency conflict with other packages

Use a virtual environment to isolate Odin from other projects: 

python -m venv odin-env
source odin-env/bin/activate
pip install odin-agent

Permission denied when installing globally

Avoid using sudo pip. Instead, install for the current user: 

pip install odin-agent --user

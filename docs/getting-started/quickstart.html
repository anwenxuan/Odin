Get Started / Quick Start

Quick Start

Up and running in 5 minutes. No API key needed to explore with mock mode.

Prerequisites

Python 3.11+

Install Odin via pip, uv, or source. See the Installation guide for details.

API Key (Optional)

Mock mode works without any key. For real AI analysis, you'll need an OPENAI_API_KEY or ANTHROPIC_API_KEY.

Mock mode simulates the full analysis pipeline using rule-based heuristics. Perfect for learning, testing, or evaluating workflows without any API costs.

Step 1: Install Odin

If you haven't installed Odin yet, do that first.

bash 
pip install -e .

Step 2: Try Without an API Key

Use mock mode to explore the full analysis pipeline. No API key required.

bash 
$ odin analyze ./examples --provider mock --verbose

Step 3: Connect Real AI

Once you're ready for full analysis, set your API key and point Odin at a real repository.

bash 
# Set your API key
export OPENAI_API_KEY=sk-your-key-here

# Analyze a repository
$ odin analyze https://github.com/owner/repo \
    --workflow vulnerability_research \
    --provider openai --model gpt-4o-mini
